Выгрузка начислений зарплаты в "1С: Бухгалтерия предприятия"

 Top  Previous  Next

 

Обработка используется для отражения начисленной заработной платы в ЗИК 11.0 (ЗИК) в 1С: Бухгалтерия предприятия (БП). Загрузка данных происходит в два этапа:

 

1. Выгрузка начисленной зарплаты в ЗИК 11.0;

2. Загрузка данных в БП;

 

Рассмотрим эти этапы поднобнее:

 

1. Выгрузка начисленной зарплаты в ЗИК 11.0;

 

•Откроем обработку выгрузки: "Начисление зарплаты" - "Сервис" - "Выгрузка начислений зарплаты в бухгалтерию":

 

 

•Выбираете период и нажимаете на кнопку "Выгрузить данные":

 

 

•Сохраним файл в папку на рабочем столе (например "Выгрузка начислений в БП"):

 

 

2. Загрузка данных в БП;

 

•Загрузка начислений в БП будет производиться через специальную обработку. Ее необходимо сохранить на рабочий стол. Для этого необходимо перейти по ссылке "Получить обработку и инструкцию по загрузке начислений из программы Инкомсофт ЗИК 11 в БП 3.0":

 

 

 

•Сохраним файл (можно в ту же папку, в которую ранее выгружали файл выгрузки начислений см. предыдущий пункт):

 

 

 

•Распакуйте файлы в папке:

 

 

 

•Запускаем 1С: Бухгалтерия предприятия. Заходим в Меню - Файл - Открыть:

 

 

•Открываем обработку, которую мы сохранили на предыдущем шаге:

 

 

•Выберите файл выгрузки начислений, который мы создали ранее (см. п. 1):

 

 

•Нажмите на кнопку "Загрузить данные":

 

 

 

Дополнительно:

 

Для того, чтобы каждый раз при выгрузке данных не сохранять обработку для загрузки данных в БП, ее можно загрузить прямо в БП, как внешнюю обработку.

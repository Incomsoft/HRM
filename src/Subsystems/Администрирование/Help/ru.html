Администрирование

Раздел предназначен для работы администратора базы данных. Содержит служебные инструменты для обслуживания базы данных, ее обновления и создания резервных копий. Так же содержит инструменты для взаимодействием с фирмой «Инкомсофт» для обеспечения технической поддержки пользователей программы.

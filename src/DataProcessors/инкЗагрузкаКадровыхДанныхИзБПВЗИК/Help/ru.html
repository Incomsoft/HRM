Данная обработка используется для загрузки начальных кадровых данных из 1С: Бухгалтерия предприятия (БП) в нашу программу.

 

Загрузка данных происходит в несколько этапов:

 

1.Получите внешнюю обработку для выгрузки кадровых данных в БП;

2.Выгрузите из БП кадровые данные;

3.Загрузите кадровые данные в ЗИК 11;

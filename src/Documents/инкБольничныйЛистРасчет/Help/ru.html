Больничный лист (расчет)

 Top  Previous  Next

 

Находится в разделе: "Начисление зарплаты" - "Документы" - "Больничный лист (расчет)".

 

Необходим для расчета суммы по больничному листу сотрудника. Содержит вкладки:

 

•Основное;

•Результат расчета;

 



Основное

 Top  Previous  Next

 

•На вкладке основное проставляем реквизиты больничного листа. Это дата выдачи, номер, серия, период;

 

 

•Если есть кадровые приказы, то вы можете загрузить в расчет информацию из кадров. Для этого нажимаем кнопку «Заполнить данные из кадров». Выбираем кадровый приказ;

•Так же устанавливается процент оплаты больничного листа в зависимости от стажа работника. Для этого в реквизите «Процент оплаты» устанавливается нужный процент;

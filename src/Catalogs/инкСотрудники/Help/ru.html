Сотрудники организации.


Содержит вкладки:

·   Основные данные;

·   Документы;

·   Данные с предыдущего места работы;

·   Дополнительно;

·   Адраса, телефоны;

 

1.1.1.   Основные данные

 

Вкладка содержит:

 

·   Базовые кадровые данные и налоговые вычеты;

·   Кадровые данные;

1.1.1.1.  Основные реквизиты и налоговые вычеты

 

·   К базовым кадровым данным относятся: ФИО сотрудника, Дата рождения, ИНН, СНИЛС и прочее. Эти данные вводятся в карточке сотрудника;

·   Налоговые вычеты отражены ввиде таблицы, в которой указан код вычета, дата действия и сумма. Сумма вычета отображается справочно;

 

1.1.1.2.  Кадровые данные

 

Содержит данные: Организация, подразделение, должность, оклад, дата приема на работу, дату увольнения и пр. Кадровые данные сотрудника отображаются в форме СПРАВОЧНО. Изменить их можно ТОЛЬКО через КАДРОВЫЕ ДОКУМЕНТЫ (ссылка на кадровые документы сотрудника находится в заголовке раздела).

 

1.1.2.   Документы

 

Содержит информацию по документу удостоверяющего личность. Содержит в себе такие реквизиты, как: вид документа, серия, номер, дата выдачи и пр.

 

1.1.3.   Данные с предыдущего места работы

 

Включает в себя данные с предыдущего места работы, такие как: облагаемый доход с начала года, общая сумма стандартных налоговых вычетов и данные по зарплате на которую были начислены страховые взносы.

 

1.1.4.   Дополнительно

 

Содержит информацию об имущественном вычете, зарплатном проекте сотрудника, комментарий к справочнику.

 

Смотрите так же:

 

·   Имущественный вычет

·   Зарплатный проект

 

1.1.4.1.  Имущественный вычет

Содержит всю необходимую информацию для учета имущественного вычета при расчете зарплаты. Информация вводится ввиде нескольких реквизитов таких как: Сумма, Код вычета, Номер уведомления, Дата выдачи и Код ФНС.

 

 

1.1.4.2.  Зарплатный проект

 

Используется для заполнения зарплатного проекта сотрудника. Данная информация используется при формировании документа "Ведомость на выплату зарплаты".

 

Содержит реквизиты: "Зарплатный проект" и "Лицевой счет".

 

 

1.1.5.   Адреса, телефоны

 

Содержит информацию по адресу прописки и по адресу проживания сотрудника. Реквизиты привязаны к справочнику ФИАС 1С. Его можно скачать с сайта 1С по адресу: https://its.1c.ru/download/fias2 при наличии подписки ИТС.
